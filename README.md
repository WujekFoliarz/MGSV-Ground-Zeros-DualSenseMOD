## USB RECOMMENDED

### Features
- Adaptive triggers
- Haptic Feedback for IDroid

### Requirements
*ViGEMBus is required! Without it the mod won't work*
https://github.com/nefarius/ViGEmBus/releases/tag/v1.22.0

### Installation
- Drop the files in the game folder

### Notes
